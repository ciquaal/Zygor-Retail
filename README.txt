1. Download the Addon:
Download the desired addon from GitHub (.zip file).


2. Extract the Addon:
Extract the downloaded .zip file to a folder on your computer.


3. Find the Addon Folder:
Default locations listed below, if you have installed your addon in an alternative location, you can use Battle.net by clicking the "Settings" button on the right hand side of "PLAY", then click "Show in Explorer".
 
Retail WoW:
C:\Program Files (x86)\World of Warcraft\_retail_\Interface\AddOns
Classic WoW:
C:\Program Files (x86)\World of Warcraft\_classic_\Interface\AddOns
Wrath Classic (WotLK):
C:\Program Files (x86)\World of Warcraft\_classic_era_\Interface\AddOns


4. Install the Addon:
Copy the extracted addon folder into the appropriate AddOns folder for your WoW version.


5. Verify and Load Addons:
Open WoW.
In the character selection screen, click "AddOns" in the bottom-left corner.
Ensure the addon is listed and enabled.
